# UpSnap Flutter Client

[日本語 README](README.ja.md)

This is a Flutter client for [UpSnap](https://github.com/seriousm4x/upsnap).

## Features

- Connect to an UpSnap server from a Flutter app
- View and manage devices
- Access UpSnap from mobile-friendly screens
- Includes Android home screen widgets
